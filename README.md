# Interactive Dashboard Demo

## Project Overview
The Interactive Dashboard Demo is a web application that allows users to visualize data interactively.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vaibhavbv2007-lab/interactive-dashboard-demo.git
   ```
2. Navigate into the project directory:
   ```bash
   cd interactive-dashboard-demo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Deployment Instructions
To deploy the application:
1. Ensure that the build is complete:
   ```bash
   npm run build
   ```
2. Deploy the build directory to your server or hosting platform.
